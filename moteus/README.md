# moteus

moteus 无刷电机控制器的核心 Python 绑定：异步控制 API（`Controller`）、多设备传输层（`Transport`/`TransportDevice`）、高频线程安全控制封装（`ControlManager`）、离线模拟器（`moteus.simulator`）等。

安装：

```bash
pip install -e .
```

图形界面工具（`tview`、`fastgui`）在独立的 [`moteus_gui`](../moteus_gui/README.md) 包中提供。

完整文档见仓库根目录 [README.md](../README.md)。
